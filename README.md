🔄 7. Updating your HTML (super easy)

Whenever you want to change it:

Go to your repo
Click your index.html
Click ✏️ Edit
Paste updated code
Click Commit changes

👉 Refresh your website — done.

⚠️ Common gotchas (important)
File must be named index.html

Use relative paths for images/CSS:

/style.css
/images/photo.jpg
If using CSS/JS files → upload them too
💡 Pro Tips
You can upload a full folder (HTML + CSS + JS + images)

You can create multiple pages:

about.html
contact.html
Then link them normally inside your HTML

FULL INSTRUCTIONS ON HOW TO SET UP START HERE
🚀 Step-by-Step: Host Your HTML on GitHub Pages
🟢 1. Create a GitHub account (if needed)
Go to https://github.com
Sign up (free)
🟢 2. Create a new repository
Click “+” (top right) → New repository

Name it something like:

html-display
Set to Public ✅ (required for free Pages)
Tick “Add a README file”
Click Create repository
🟢 3. Upload your HTML file
Inside your repo, click “Add file” → “Upload files”
Drag in your HTML file
👉 Important:

Rename it to:

index.html

(this makes it load automatically)

Click Commit changes
🟢 4. Enable GitHub Pages
Go to Settings (top menu in your repo)
Scroll to “Pages”
Under Source:
Select: Deploy from a branch
Branch: main
Folder: / (root)
Click Save
🟢 5. Get your live URL

After ~30 seconds, you’ll see something like:

https://yourusername.github.io/html-display/

👉 Open it — your HTML should display exactly as designed.

🌐 6. Add it to Elementor

Now embed it in your site using an iframe:

<iframe src="https://yourusername.github.io/html-display/" 
        width="100%" 
        height="800px" 
        style="border:none;">
</iframe>
